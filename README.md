## Drag & Drop with a Playful Twist!
DropBoxy is a modern, drag-and-drop image uploader with a playful twist. Easily upload, preview, and manage your images in a responsive web interface. Perfect for portfolios, web apps, or just organizing your snaps!

---

## Features

- Drag & Drop images directly into the uploader  
- Browse and select files manually  
- Real-time preview of uploaded images  
- Dynamic image counter  
- Remove individual images or clear all at once  
- Smooth progress bar animation during uploads  
- Fully responsive layout for all screen sizes  
- Scrollable image grid for easy navigation
